Football Live Scoreboard — Live Scores, Leagues & Match Stats (React + Node.js)

A production-ready full-stack football live score application powered by API-Football v3. Includes a polished React frontend with dark mode, league browser, match detail pages with goals/events/statistics, and a Node.js/Express backend with caching and Socket.IO live polling.

IMPORTANT: API KEY NOT INCLUDED
Buyers install the source code and add their own free API-Football key in Backend/.env.
No shared API is provided — each user registers at api-football.com and uses their personal key.

FEATURES
• Live match scores and status updates
• Today's fixtures grouped by league
• Searchable league browser (Premier League, Champions League, La Liga, and more)
• Match detail: goals, assists, events timeline, possession, shots, cards
• Dark mode + customizable stats display
• Mobile-first responsive UI (Tailwind CSS)
• Backend in-memory caching for API rate limits
• Clean REST API architecture

TECH STACK
• Frontend: React 18, Vite, Tailwind CSS, React Router
• Backend: Node.js, Express, Socket.IO, Axios
• Data: API-Football v3 (free tier supported)

SETUP (5 minutes)
1. cd Backend → npm install → copy .env.example to .env → paste YOUR API key → npm run dev
2. cd Frontend/Football-Frontend → npm install → npm run dev
3. Open http://127.0.0.1:5173
See API-KEY-SETUP.txt inside the zip for detailed steps.

Perfect for developers, agencies, and startups building a sports scores product or learning full-stack React + Node.js integration.
